JittaKey에 오신 것을 환영합니다

일본어 입력을 더욱 쉽고 빠르게 만들어주는 혁신적인 입력 방식입니다.

🔑 주요 기능

  • 로마자 입력 대비 약 20% 입력 횟수 절감
  • WYSIWYG 방식 – 보이는 그대로 입력
  • 자주 쓰이는 문자 1타 입력 (예: い단, 작은 모음, っ, ん, ゃ, ゅ, ょ 등)
  • 사용되지 않는 키에 연상 기반 문자 배정 (예: Q = ん, F = を, J = ゆ 등)
  • 앱과 커스텀 키보드 모두에서 사용 가능

🧠 Mnemonic Mapping for Unused Keys

  • Q → 「ん」: Queueing의 “Q”에서 “ing”＝「ん」를 연상할 수 있음
  • F → 「を」: 카타카나의「ヲ」와F가 데칼코마니임
  • J → 「ゆ」: 유도（Juudou）에 “ゆ”음이 포함되어 있음
  • L → 「よ」: YOLO에서 「よ」음과 L이 포함되어 있음
  • X → 「―」: X-mas（크리스마스의 축약형）에 X와 하이픈이 포함되어 있음
  • C → 「っ」: “CHU”의 시작이 C여서, 「っ」를 연상할 수 있음
  • V → 「ヴ」: 영어의 “V”를 표현한 카타카나임

ℹ️ 특별한 입력 규칙

「きょ」는 き + ょ로 입력합니다 – 키보드에 보이는 그대로입니다.

「きあ」를 입력하고 싶다면 kia처럼 입력해야 합니다 (ka는 「か」로 인식됩니다). 이 규칙은 kii = きい, kiu = きう, kie = きえ, kio = きお에도 적용됩니다.
